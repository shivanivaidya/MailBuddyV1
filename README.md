# MailBuddyV1

MailBuddyV1 is a Gmail semantic intelligence layer. It turns real inbox patterns into source-backed tasks, updates, semantic threads, financial items, search results, and Attention Today workflows.

This repository is currently in the spec-driven planning phase. There is no application scaffold yet.
